# Changelog: v2

Detailed release history for the `2.x` line.

## [2.0.0] - 2024-XX-XX

### Changed

- Breaking: moved to a zero-dependency architecture with `mongoose` as a peer
  dependency.
- Marked the line as the first production-ready major release.

### Added

- Built-in plugin suite, including timestamping, soft delete, caching, audit,
  and validation helpers.
- QueryParser with security hardening.
- JSON Schema generation for OpenAPI / Fastify-style usage.
